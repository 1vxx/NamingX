## 重新定义命名

在 NamingX，我们相信每一个变量名都不仅仅是一个标识符，它是思想的载体，是代码逻辑的基石。

长期以来，开发者们在“命名”这一计算机科学的终极难题前徘徊。我们问自己：**不仅仅是生成名字，我们能否让这个过程变得令人愉悦？我们能否在指尖注入一丝魔力？
**

于是，有了 NamingX。

它不仅仅是一个插件，它是你与代码对话的方式。

## 核心体验

### 🔮 智能，触手可及

无需繁琐的操作。输入你的想法，NamingX 会深入理解语境，利用最先进的 AI 模型，为你呈现最精准、最优雅的变量命名建议。这不仅是自动化，这是直觉的延伸。

### 🎨 极简美学，原生体验

我们极其在乎设计。NamingX 完美融合进你的 IDE，原生般的视觉体验，每一个像素都经过精心打磨。它就在那里，安静而强大，只在你需要的时候出现。

### 😄 代码中的幽默感

谁说编程必须是枯燥的？每一个生成的建议都附带着一段简短、幽默且富有哲理的中文解释。这不只是为了解释，更是为了让你在深夜 debug
时会心一笑。

### 🛠 你的工具，由你定义

我们相信赋予用户控制权。

- **自定义模型**：支持配置 OpenAI 兼容的 API 和模型名称。
- **连接测试**：一键测试连接性，确保时刻在线。
- **格式多样**：CamelCase, snake_case... 我们懂你需要什么。

## 快速开始

1. **安装**：从插件市场下载 NamingX。
2. **打开**：在侧边栏找到那个标志性的灯泡图标。
3. **配置**：进入设置，填入你的 API Key。
4. **体验**：输入描述，点击生成。Boom。魔术发生了。

我们去除了所有不必要的干扰，只留下了最纯粹的功能。这不仅仅是效率的提升，更是心智的解放。
